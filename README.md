# Real Estate Web Application

A fully functional Real Estate Web Application developed using HTML, CSS, and JavaScript.

This application enables users to list and explore properties such as houses for rent, flats for sale/rent, and land for sale.

## Features
- Role-based navigation (Owner / Tenant / Buyer)
- Property listing and browsing
- Image upload functionality
- Area-based search
- Dynamic data handling using localStorage

## Description
This project simulates a real-world property listing platform where users can add and explore properties easily through a simple and interactive interface.

## Technologies Used
- HTML
- CSS
- JavaScript
